Ин Figma тарҳи як лендинг-пейҷ (Landing Page) барои мактаби барномасозии YtYt мебошад.
Сайт барои муаррифии курсҳои омӯзишии барномасозӣ сохта шуда, ба корбарон маълумот дар бораи технологияҳои омӯзишшаванда,
раванди таълим, бартариҳои курс ва нархнома пешниҳод мекунад. Дар тарҳ инчунин бахшҳои дастгирии донишҷӯён, методикаи омӯзиш, ҷадвали арзиши курсҳо ва маълумоти тамос ҷойгир шудаанд.
Ҳадафи асосии ин вебсайт ҷалби донишҷӯёни нав ва шинос кардани онҳо бо имкониятҳои омӯзишии марказ мебошад.
